# 6keyKEEB

Tired of boring scrolling? Use the 6keyKEEB, with buttons for page up, page down, and all the arrow keys! how nice

## BOM (Bill of Materials)

- 2 SK6812 MINI-E LED
- 6 Blank DSA black keycaps
- 4 M3x16mm screws
- 1 Seeed XIAO RP2040
- 6 MX-Style switches
